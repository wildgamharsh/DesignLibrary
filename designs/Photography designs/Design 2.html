Template notes · Editing guide

Gallery items
  • Duplicate .gallery-item blocks to add images.
  • Update data-category with: wedding, portrait, editorial, film, landscape.
  • Edit data-title, data-caption, and data-location for lightbox content.
Colors & type
  • Adjust the palette via :root variables (e.g. --accent, --bg-main).
  • Swap fonts globally by changing font-family on body.
  • Buttons share the .btn base class with .btn-primary / .btn-ghost.
Layout blocks
  • Each major section (hero, services, gallery, projects, about, testimonials, contact) is modular.
  • Reorder by moving entire <section> elements.
  • Mobile behavior is tuned at 768px and 520px breakpoints in the media queries.
